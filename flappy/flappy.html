Flappy

Flappy is a web-based side-scroller where the player controls a small yellow circle attempting to jump/fly between pipes to stay alive and achieve the highest score possible. Flappy is inspired by the old mobile side-scroller "Flappy Bird". The game uses real life physics to make the game feel smooth and provide a challenge. The game can be played on only a keyboard, but it is also designed to work with an ELEGOO UNO R3 arduino. The code supports a button and a LED meant to control the bird and provide feedback to the arduino of which state the game is in.

Graphics

The game utilizes the p5 javascript library to draw and perform all of the game mechanics and functions. Their are two classes, Player and Obstacle, used to control the objects in the game. There are two functions, keyPressed() and buttonPress(), which handle the user inputs. Function keyPressed() is a built in p5 function which detects keyboard input to handle the two keyboard controls:

  • Space Bar - Jump/Resume/Start
  • Escape Key - Pause

The player physics use the kinematic formulas:

  • v = v0 + at
  • xt = x0 + vt

Where a = 20 px/s and the t = 1/60 (60 fps).

The obstacles are objects in a holder array so they are easy to be operated on. They are spawned in order with a set distance between each other, but they are always randomized in height.The player scores when they pass the midpoint of the pipe, and the pipe continues moving all the way to the left side of the canvas. Once a pipe completely exits the left side of the canvas, it's position will be set to the last pipe on the right and the height of the gap will be randomized so the user does not go through the "same" pipe. By designing the obstacles this way, the game only uses the minimum number of obstacles to span the screen with no gaps to create an endless game with no repeated patterns.

Sound

Flappy utilizes the Tone javascript library to use synthesizers and players to generate sound effects and music. The music is synthesized using two of Tone's PolySynths, one for bass line and one for melody. The bass and melody play during the play state, but when the player dies, the melody stops. When the game is paused or waiting to start, no music plays. The song is based off of the popular 80s hit "Jump" by Van Halen.

Sound effects are achieved through a Tone Player. The player gets triggered during game events to play the intended sound. Specifically, the flap sound is triggered when the player jumps, and the bonk sound is triggered when the player collides with a pipe or the ground. The sound effects are from freesound.org:

  • Flap
  • Bonk

Hardware

The game was developed with keyboard and mouse controls for regular use/play. There is also support for arduino controls. Using an ELEGOO UNO R3 arduino, the controls were ported to a singular button. The arduino also harnessed an LED used to display what state the game was in.

  • Digital Input: Button connected to pin 2
  • Digital Output: LED connected to pin 9
